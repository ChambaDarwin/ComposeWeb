Firebase

Domina el desarrollo moderno de Android

  • Crud Realtime Database

    Aprende a crear un CRUD completo utilizando Firebase Realtime Database. Descubre cómo insertar, leer, actualizar y eliminar datos en una base de datos No-SQL, donde la información se almacena mediante pares clave-valor y se sincroniza en tiempo real.

  • Crud de Imagenes

    Aprenderas como implementar un CRUD de imágenes en Android. Guardamos las imágenes en el almacenamiento interno del dispositivo, obtenemos su ruta completa, la registramos en Firebase Realtime Database y luego mostramos la imagen en la app.

Como crear una unica instacia de supabaseClient

la injeccion de dependencias es un patron de diseño que consiste en no crear los objetos que necesita dentro de una clase si no que alguien externo se los inyecte

vamos a proveer una unica instacia de supabaseClient para ello realizaremos lo siguiente 


    @Module
@InstallIn(SingletonComponent::class)
object Modulo {

    @Provides
    @Singleton
    fun providerSupabaseClient(): SupabaseClient {
        return createSupabaseClient(
            supabaseUrl = "",
            supabaseKey = ""
        ) {
            install(Storage)
        }

    }
}

Crear un objeto de supabaseClient

lo primero que tenemos que hacer es obtener el projecturl y el api key para ello nos dirijimos al final a project settings ->data api y obtenes el parametro supbaseUrl luego nos dirijimos a api key para completar el parametro supabaseKey

Trabajando con el viewModel

Los metodos que trabajan con el Storage deben utilizar en funciones suspendidas o dentro del viewModelScaope.launch{}y todos los metodos se utiliza con el objeto en este caso el objeto se llama supabase.storage.from("image") es el nombre del bucket

Como subir una imagen

para subir una imagen utilizamos el metodo upload () que recibe dos parametros el path que es el nombre de la imagen y data que es el ByteArray que contiene la imagen

Como obtener la public url

con la public url puedes utilizar para mostrar una imagen o guardar en una base de datos para ello utiliza el metodo publicUrl que recibe como parametro el path es decir el nombre de la imagen 


            supabase.storage.from("image").publicUrl(path)

Como recuperar todas las imagenes del Storage

para recuperar las imagenes del storage utilizamos el metodo list() este metodo nos devuelve una lista de tipo BucketItem pero aun no tenemos la publicUrl para ello utilizmos el name que nos devuelve el path de la imagen


        val lista=   supabase.storage.from("image").list().map {
                supabase.storage.from("image").publicUrl(it.name)
            }

Como eliminar una imagen

para eliminar una imagen utilizamos el metodo delete() que recibe como parametro el nombre de la imagen pero recuerda que la imagen es algo como esto https://ogdtmenhbheftlkzbpxd.supabase.co/storage/v1/object/public/image/img_1770229779832.jpg pero lo que nosostros necesitamos es solo esto img_1770229779832.jpg entonces para ello utilizaremos lo siguiente


        supbase.storage.from("image").delete(image.substringAfterLast("/"))

substringAfterLast crea una subcadena despues del ultimo "/" y a partir de ahi ya podemos obener el nombre de la imagen y por ende podemos eliminar

Como abrir la Galeria y seleccionar una foto

para abrir la galeria utilizaremos rememberLauncherForActivityResult que se utiliza para abrir una actividad del sistemas por ejemplo pedir permisos ,abrir la camara,tomar una foto, o abrir la galeria , tambien podemos observar que recibe como parametro un contrato el contrato define que accion va a realizar el launcher y que tipo de resultado se va a devolver

Ejemplo de contratos

Contrato ¿Qué hace? Input Output
GetContent() Seleccionar archivo String (mime) Uri?
PickVisualMedia() Fotos/Video PickVisualMediaRequest Uri?
TakePicture() Cámara Uri Boolean
RequestPermission() Permiso String Boolean
StartActivityForResult() Intent Intent ActivityResult

Regla

Contrato = accion + tipo de resultado

Uri es un identificador unico de un recurso que te acceso temporal para leerlo

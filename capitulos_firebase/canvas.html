📦 Dependencias Necesarias

Agrega las siguientes dependencias en el archivo build.gradle a nivel de módulo (app):

dependencies {
    // Compose Foundation para gráficos
    implementation("androidx.compose.foundation:foundation:1.6.0")

    // Compose UI (requerido para Canvas)
    implementation("androidx.compose.ui:ui:1.6.0")

    // Compose UI Graphics (para Paint, Path, etc.)
    implementation("androidx.compose.ui:ui-graphics:1.6.0")
}
💡 Nota: Si trabajas con Views tradicionales de Android (no Compose), solo necesitas el SDK de Android estándar ya que android.graphics.Canvas viene incluido.

🎨 Conceptos Fundamentales

1. Crear un Bitmap (Lienzo)

El Bitmap es la superficie donde dibujaremos nuestra imagen. Piensa en él como un lienzo en blanco con dimensiones específicas y calidad de color.

val bitmap = Bitmap.createBitmap(800, 600, Bitmap.Config.ARGB_8888)
Parámetro Descripción
800 Ancho en píxeles
600 Alto en píxeles
ARGB_8888 Configuración de color (32 bits: 8 bits para Alpha, Red, Green, Blue)

2. Crear un Canvas (Herramienta de Dibujo)

El Canvas es el objeto que ejecuta todas las operaciones de dibujo sobre el Bitmap. Recibe el bitmap como parámetro.

val canvas = Canvas(bitmap)
⚠️ Importante: Debes utilizar el import correcto:
import android.graphics.Canvas

3. Crear un Paint (Pincel/Estilo)

El objeto Paint define las características de dibujo: color, grosor, tipo de fuente, etc. Puedes crear múltiples objetos Paint con diferentes estilos.

val lapizRojo = Paint().apply {
    color = Color.RED
    textSize = 30f
    isAntiAlias = true
}

📐 Sistema de Coordenadas

En Android Canvas, el sistema de coordenadas funciona de la siguiente manera:

💡 Puntos clave:
  • El origen (0,0) está en la esquina superior izquierda
  • X aumenta hacia la derecha
  • Y aumenta hacia abajo (contrario al plano cartesiano tradicional)

✏️ Dibujar Texto

El método canvas.drawText() dibuja texto en el canvas. Recibe cuatro parámetros:

canvas.drawText("Hola Mundo", 200f, 400f, paintTexto)

🎨 Propiedades del Paint

Color

Define el color con el que se dibuja (texto, líneas, formas):

paint.apply {
    color = android.graphics.Color.RED
    // o usando código hexadecimal
    color = android.graphics.Color.parseColor("#FF5733")
}

Tamaño de Texto

textSize = 30f  // En píxeles

Tipo de Fuente (Typeface)

// Fuentes del sistema
typeface = Typeface.MONOSPACE
typeface = Typeface.SERIF
typeface = Typeface.SANS_SERIF

// Fuentes personalizadas
typeface = Typeface.create("Arial", Typeface.BOLD)
typeface = Typeface.create("Arial", Typeface.ITALIC)
typeface = Typeface.create("Arial", Typeface.BOLD_ITALIC)

Grosor de Línea

strokeWidth = 10f  // En píxeles

Estilo de Relleno

style = Paint.Style.FILL            // Relleno sólido
style = Paint.Style.STROKE          // Solo contorno
style = Paint.Style.FILL_AND_STROKE // Ambos
💡 Tip: Recuerda definir un color antes de usar Paint.Style.FILL

Alineación de Texto

textAlign = Paint.Align.CENTER  // Centrado
textAlign = Paint.Align.LEFT    // Izquierda (predeterminado)
textAlign = Paint.Align.RIGHT   // Derecha

Espaciado entre Letras

letterSpacing = 0.3f   // Aumenta el espacio
letterSpacing = -0.05f // Reduce el espacio

Suavizado de Bordes (Anti-Aliasing)

isAntiAlias = true  // Bordes más suaves y profesionales

▭ Dibujar Rectángulos

El método drawRect() crea un rectángulo definido por sus bordes:

canvas.drawRect(left, top, right, bottom, paint)

// Ejemplo
canvas.drawRect(100f, 100f, 400f, 200f, paintRectangulo)
Parámetro Descripción
left Borde izquierdo (coordenada X donde inicia)
top Borde superior (coordenada Y donde inicia)
right Borde derecho (coordenada X donde termina)
bottom Borde inferior (coordenada Y donde termina)
💡 Cálculo de dimensiones:
  • Ancho: right - left
  • Alto: bottom - top
  • Ejemplo: Si top = 100f y bottom = 200f, la altura será 100f

📐 Mediciones de Texto

Obtener el Ancho de un Texto

Usa measureText() del objeto Paint para conocer el ancho que ocupará un texto:

val paint = Paint().apply {
    textSize = 30f
}

val ancho = paint.measureText("Hola Mundo")
println("El texto ocupa: $ancho píxeles de ancho")

Calcular la Altura de un Texto

Utiliza fontMetrics para obtener información sobre la altura del texto:

val paint = Paint().apply {
    textSize = 30f
}

val metrics = paint.fontMetrics
val altura = metrics.descent - metrics.ascent

println("Altura del texto: $altura píxeles")
⚠️ Importante sobre ascent:
El valor ascent es negativo (representa la parte del texto que va hacia arriba desde la línea base). Por eso se resta: descent - ascent. Debido a la ley de signos, esto efectivamente suma ambos valores absolutos.
💡 Uso práctico: Estas mediciones son esenciales para:
  • Centrar texto dinámicamente
  • Calcular espacios entre líneas
  • Ajustar el tamaño del bitmap según el contenido
  • Crear layouts responsivos

📤 Compartir la Imagen por WhatsApp

Paso 1: Configurar FileProvider

Agrega lo siguiente en tu archivo AndroidManifest.xml dentro de la etiqueta <application>:

<provider
    android:name="androidx.core.content.FileProvider"
    android:authorities="${applicationId}.provider"
    android:exported="false"
    android:grantUriPermissions="true">
    <meta-data
        android:name="android.support.FILE_PROVIDER_PATHS"
        android:resource="@xml/file_paths"/>
</provider>

Paso 2: Crear archivo de rutas

Crea un directorio xml en res (si no existe) y dentro crea el archivo file_paths.xml:

<?xml version="1.0" encoding="utf-8"?>
<paths>
    <cache-path name="images" path="." />
</paths>

Paso 3: Guardar y compartir el Bitmap

fun compartirPorWhatsApp(context: Context, bitmap: Bitmap) {
    try {
        // Guardar bitmap en caché
        val cachePath = File(context.cacheDir, "images")
        cachePath.mkdirs()
        
        val file = File(cachePath, "imagen_compartida.png")
        val fileOutputStream = FileOutputStream(file)
        bitmap.compress(Bitmap.CompressFormat.PNG, 100, fileOutputStream)
        fileOutputStream.close()
        
        // Crear URI con FileProvider
        val imageUri = FileProvider.getUriForFile(
            context,
            "${context.packageName}.provider",
            file
        )
        
        // Crear intent para compartir
        val intent = Intent(Intent.ACTION_SEND).apply {
            type = "image/png"
            putExtra(Intent.EXTRA_STREAM, imageUri)
            addFlags(Intent.FLAG_GRANT_READ_URI_PERMISSION)
            setPackage("com.whatsapp") // Específico para WhatsApp
        }
        
        context.startActivity(Intent.createChooser(intent, "Compartir vía"))
        
    } catch (e: Exception) {
        e.printStackTrace()
        Toast.makeText(context, "Error al compartir", Toast.LENGTH_SHORT).show()
    }
}
💡 Nota: Si quieres permitir otras apps además de WhatsApp, simplemente elimina la línea setPackage("com.whatsapp")

📝 Ejemplo Completo: Ticket de Venta

fun crearTicketVenta(productos: List<Producto>): Bitmap {
    // Calcular altura dinámica
    val alturaBase = 500f
    val alturaPorProducto = 60f
    val alturaTotal = (alturaBase + productos.size * alturaPorProducto).toInt()
    
    val bitmap = Bitmap.createBitmap(800, alturaTotal, Bitmap.Config.ARGB_8888)
    val canvas = Canvas(bitmap)
    
    // Paint para fondo
    val paintFondo = Paint().apply {
        color = Color.WHITE
        style = Paint.Style.FILL
    }
    
    // Paint para título
    val paintTitulo = Paint().apply {
        color = Color.BLACK
        textSize = 50f
        textAlign = Paint.Align.CENTER
        typeface = Typeface.create("Arial", Typeface.BOLD)
        isAntiAlias = true
    }
    
    // Paint para productos
    val paintProductos = Paint().apply {
        color = Color.BLUE
        textSize = 25f
        typeface = Typeface.MONOSPACE
        isAntiAlias = true
    }
    
    // Dibujar fondo
    canvas.drawRect(0f, 0f, 800f, alturaTotal.toFloat(), paintFondo)
    
    // Dibujar título
    canvas.drawText("TICKET DE VENTA", 400f, 100f, paintTitulo)
    
    // Dibujar productos
    var y = 200f
    productos.forEach { producto ->
        canvas.drawText(producto.nombre, 50f, y, paintProductos)
        canvas.drawText("$${producto.precio}", 600f, y, paintProductos)
        y += 60f
    }
    
    return bitmap
}

✅ Mejores Prácticas

  • Siempre usa isAntiAlias = true para textos más nítidos
  • Calcula el tamaño del bitmap dinámicamente según el contenido
  • Recicla bitmaps que ya no uses: bitmap.recycle()
  • Usa measureText() para layouts precisos
  • Comprime con calidad 90-100 para PNG de alta calidad
  • Maneja excepciones al compartir archivos

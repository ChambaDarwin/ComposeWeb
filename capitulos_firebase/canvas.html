Como crear una imagen con Canvas

lo primero que haremos sera definir la superficie donde vamos a dibujar y el tamaño y la calidad de la imagen para ello creamos un objeto de tipo BitMap que recibe tres parametros el primero es el ancho el segundo es la altura y el tercero es la calidad de la imagen


        val bitmap = createBitMap(800,600,BitMap.Config.ARGB_8888)

Como crear un objeto Canvas

este objeto ejecuta todas las operaciones que se va a dibujar en el Bitmap por ende recibe como parametro el objeto bitmap que creamos en el paso anterior, entonces creamos un objeto de la siguiente forma


        val canvas = android.grapics.Canvas(bitmap)

Importante

debes utilizar el import android.grapics.Canvas

Como crear un objeto Paint()

un objeto paint es como crear un lapiz personalizado con color , grosor etc y puedes crear muchos objetos Paint como desees de la siguiente forma


    val lapizRojo = Pain().apply{
        color=Color.Red
    }

Como escribir texto

recuerda que el objeto canvas es el que ejecuta las operaciones de dibujo sobre el bitmap es decir que es la persona que dibuja entonces utilizamos el metodo canvas.drawText() este recibe cuatro parametros el primer parametro es el texto que deseas mostrar el segundo es la posicion en x el tercer parametro es la posicion en y el utltimo parametro es un objeto Paint que contiene todas las caracteristicas

para escribir utilizamos coordenadas solo recuerda como se trabaja en un plano cartesiano es practicamente igual

Propiedades importantes del Paint()

recuerda que el paint es el pincel con el que dibujamos

color: Define el color con el que se dibuja: texto, lineas


    color = android.grapics.Color.RED
    color =android.grapics.Color.parseColor("#FFFFF")

Texto: Controla el tamaño del texto


textSize= 30f

typeFace


typeFace= Typeface.MONOSPACE
typeFace = Typeface.SERIF
//fuentes personalizadas
//recibe como segundo parametro un entero asi que asegurante de elegir una opcion que te devuelva un nuemero entero
typeFace = Typeface.create("Arial",Typeface.Bold)

strokeWidth: Grosor de la linea


    strokeWidth = 10f

style:Rellena formas con colores solidos


    //importante agregar un color
    style = Paint.Style.FILL
    style =Paint.Style.STROKE //solo el contorno
    style = Paint.Style.FILL_AND_STROKE //relleno y contorno

textAlign: alineacion del texto


textAlign = Paint.TextAlign.CENTER
textAlign = Paint.TextAlign.LEFT
textAlign = Paint.TextAlign.RIGHT

letterSpacing: Espacio entre letras


    letterSpacing = 0.3f
    letterSpacing = -0.05f

isAntiAlias :suavisado de bordes


    isAntialias = true

Como crear lineas

para crear una linea utilizamos el metodo drawLine(startx,starty,endx,endy,Paint) recibe cuatro parametros recuerda que x trabaja en el eje horizontal y y en el eje vertical startx y starty te dice en el eje horizontal donde inicia y donde finaliza de igual forma starty y end y en el eje vertical donde inicia y donde termina


    canvas.drawLine(startx,starty,endx,endy,Paint)
    canvas.drawLine(200f,200f,200f,600f,Paint)

Como crear un rectangulo

para crear un rectuangulo se utiliza el metodo canvas.drawRect y recibe 4 5 parametros el primer pararametro left le indicamos donde inicia y con right le indicamos donde finaliza y de igual forma top le indicamos donde inicia y bottom donde finaliza es decir que left y right trabajan en el eje horizontal y top y bottom en el eje vertical y el utlimo parametro sera el Paint

Importante

para definir la altura del

para definir la altura del rectangulo se debe restar el bottom - top ejemplo si el top le indicas que inicie en 100f y el bottom le indicas que finalice en 150f entonces tendras una altura de 50f 


    canvas.drawRect(left,top,right,bottom,Paint)
    canvas.drawRect(0f,100f,1080f,150f,paint)

    canvas.drawRect(100f,100f,400f,150f,paint)

Obtener el ancho de una palabra

para obtener el ancho de una palabra lo que debes hacer es utilizar el objeto Paint que contiene todas las caracteristicas y utilizar el metodo measureText que recibe como parametro la candena de texto que deseas conocer el ancho 


    val metrics = Paint.measureText("hola")

Como calcular el alto de una palabra

para obtener el alto real de una palabra de igual forma necesitas el objeto Pain y utilizar el metodo ascendt y descent


    val metrics = Paint.fontMetrics 
    val alto = metrics.descent - metrics.ascent

Importante

ascent es un valor negativo es por eso que se le resta y de esa forma se suma por la ley de signos

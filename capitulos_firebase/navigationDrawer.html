📖 ¿Qué es un ModalNavigationDrawer?

El ModalNavigationDrawer es un componente de Material Design 3 que proporciona un panel lateral deslizable para la navegación de tu aplicación. Sus características principales son:

🎨 Modal

Oscurece el contenido principal cuando está abierto, enfocando la atención del usuario en el menú.

⚡ Deslizable

Se puede abrir deslizando desde el borde izquierdo de la pantalla o mediante un botón.

🔒 Bloqueante

Bloquea la interacción con la pantalla principal hasta que se cierre o seleccione una opción.

📱 Responsivo

Se adapta automáticamente a diferentes tamaños de pantalla y orientaciones.

💡 Consejo: El Navigation Drawer es ideal para aplicaciones con múltiples secciones que no necesitan estar visibles constantemente.

🛠️ Implementación Básica

1. Estado del Drawer

Primero, necesitas crear un estado para controlar si el drawer está abierto o cerrado:

@Composable
fun MyApp() {
    val drawerState = rememberDrawerState(DrawerValue.Closed)
    val scope = rememberCoroutineScope()
    
    // ... resto del código
}
Explicación:
  • rememberDrawerState: Crea y recuerda el estado del drawer entre recomposiciones
  • DrawerValue.Closed: Estado inicial (también puede ser DrawerValue.Open)
  • rememberCoroutineScope(): Necesario para ejecutar operaciones de suspensión como abrir/cerrar el drawer

2. Estructura del ModalNavigationDrawer

ModalNavigationDrawer(
    drawerState = drawerState,
    drawerContent = {
        ModalDrawerSheet {
            // Contenido del menú lateral
            Text("Menú", modifier = Modifier.padding(16.dp))
            Divider()
            NavigationDrawerItem(
                label = { Text("Inicio") },
                selected = false,
                onClick = { /* Navegar */ }
            )
            NavigationDrawerItem(
                label = { Text("Perfil") },
                selected = false,
                onClick = { /* Navegar */ }
            )
        }
    }
) {
    // Contenido principal de la pantalla
    Scaffold(
        topBar = { /* TopBar */ }
    ) { paddingValues ->
        // Tu contenido aquí
    }
}

🎨 Personalización Avanzada

Drawer Personalizado con Header

@Composable
fun CustomDrawerContent(
    user: User,
    onMenuClick: (String) -> Unit,
    onClose: () -> Unit
) {
    ModalDrawerSheet(
        modifier = Modifier.width(280.dp)
    ) {
        // Header con foto de perfil
        Column(
            modifier = Modifier
                .fillMaxWidth()
                .background(MaterialTheme.colorScheme.primary)
                .padding(24.dp)
        ) {
            Image(
                painter = painterResource(R.drawable.profile),
                contentDescription = "Foto de perfil",
                modifier = Modifier
                    .size(72.dp)
                    .clip(CircleShape)
            )
            Spacer(modifier = Modifier.height(16.dp))
            Text(
                text = user.name,
                style = MaterialTheme.typography.titleMedium,
                color = Color.White
            )
            Text(
                text = user.email,
                style = MaterialTheme.typography.bodySmall,
                color = Color.White.copy(alpha = 0.8f)
            )
        }

        Divider()

        // Elementos del menú
        NavigationDrawerItem(
            icon = { Icon(Icons.Default.Home, null) },
            label = { Text("Inicio") },
            selected = false,
            onClick = {
                onMenuClick("home")
                onClose()
            },
            modifier = Modifier.padding(horizontal = 12.dp)
        )
        
        NavigationDrawerItem(
            icon = { Icon(Icons.Default.Person, null) },
            label = { Text("Perfil") },
            selected = false,
            onClick = {
                onMenuClick("profile")
                onClose()
            },
            modifier = Modifier.padding(horizontal = 12.dp)
        )

        Spacer(modifier = Modifier.weight(1f))

        Divider()
        
        // Cerrar sesión al final
        NavigationDrawerItem(
            icon = { Icon(Icons.Default.ExitToApp, null) },
            label = { Text("Cerrar sesión") },
            selected = false,
            onClick = { onMenuClick("logout") },
            modifier = Modifier.padding(horizontal = 12.dp),
            colors = NavigationDrawerItemDefaults.colors(
                unselectedTextColor = Color.Red,
                unselectedIconColor = Color.Red
            )
        )
    }
}

⚙️ Funciones Útiles

Abrir el Drawer

scope.launch {
    drawerState.open()
}

Cerrar el Drawer

scope.launch {
    drawerState.close()
}

✅ Mejores Prácticas

  1. Usar CoroutineScope: Siempre usa rememberCoroutineScope() para las operaciones del drawer.
  2. Cerrar automáticamente: Cierra el drawer después de seleccionar una opción de navegación.
  3. Feedback visual: Marca el elemento seleccionado en el menú.
  4. Accesibilidad: Siempre proporciona contentDescription a los iconos.
  5. Ancho consistente: Usa un ancho fijo (entre 256dp y 320dp) para el drawer.
  6. Gestos: Permite tanto el botón como el gesto de deslizar para abrir el drawer.
⚠️ Importante: No uses el Navigation Drawer para navegación primaria en tablets o pantallas grandes. En esos casos, considera usar NavigationRail o NavigationBar permanente.

🐛 Problemas Comunes y Soluciones

1. El drawer no se cierra al tocar fuera

Solución: Esto es el comportamiento esperado del ModalNavigationDrawer. Si quieres cerrarlo automáticamente, usa:

ModalNavigationDrawer(
    drawerState = drawerState,
    gesturesEnabled = true, // Permite cerrar con gesto
    // ...
)

2. Crash al intentar abrir el drawer

Solución: Asegúrate de usar scope.launch para operaciones de suspensión:

// ❌ Incorrecto
onClick = { drawerState.open() }

// ✅ Correcto
onClick = {
    scope.launch {
        drawerState.open()
    }
}

3. El contenido se superpone con el drawer

Solución: Asegúrate de que el contenido principal esté dentro del bloque content del ModalNavigationDrawer, no dentro de drawerContent.

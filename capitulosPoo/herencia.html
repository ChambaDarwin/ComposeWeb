¿Qué es la Herencia?

classDiagram class Persona { -nombre : String -edad : int +Persona(edad : int) +getNombre() : String +getEdad() : int } class Estudiante { -codigo : String +Estudiante(edad : int, codigo : String) +getCodigo() : String +estudiar() : void } Persona <|-- Estudiante

La herencia permite a una clase hija heredar los atributos y métodos de la clase padre. Además, podemos crear nuevas clases a partir de una clase base y podemos agregar y modificar comportamientos sin necesidad de reescribir código.

En Java le indicamos que vamos a heredar utilizando la palabra reservada extends, por ejemplo:

class Persona {
    int edad = 23;
}

class Estudiante extends Persona {
    
}

// En el método main
Estudiante es = new Estudiante();
System.out.println(es.edad); // resultado: 23

En este caso la clase Persona no tiene un constructor y hasta ahora funciona correctamente. Lo siguiente que haremos será crear un constructor en la clase base.

Importante

Regla fundamental: Cuando una clase hereda de otra, se debe invocar al constructor de la clase padre para inicializar los atributos heredados antes de invocar al constructor de la clase hija.

Para invocar al constructor de la clase padre se realiza utilizando la palabra reservada super.

Utilizamos el constructor de la clase hija para pasar los parámetros que necesita la clase base para ser inicializada:

class Estudiante extends Persona {
    
    public Estudiante(int edad) {
        super(edad);
    }
}

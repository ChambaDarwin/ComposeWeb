☕ Clases y Objetos

Fundamentos de la Programación Orientada a Objetos

1¿Qué es una Clase?

Una clase es una plantilla o molde que define los atributos y métodos de un objeto. Las propiedades representan las características principales de un objeto, mientras que los métodos son las acciones que el objeto puede realizar.

En esta práctica crearemos una clase con atributos y métodos, y luego imprimiremos su información. A continuación se describe la clase que vamos a trabajar:

classDiagram class Persona { - String nombre - int edad - String email + saludar() + cumplirAnios() }

2Cómo Crear una Clase

class Persona {
    
}

Para crear una clase utilizamos la palabra reservada class, seguida del nombre de la clase (en este caso Persona). Luego abrimos y cerramos llaves { }. Dentro de las llaves definimos las propiedades y métodos de la clase.

💡 Importante: Una palabra reservada es aquella que tiene un significado especial dentro del lenguaje y no puede ser utilizada como nombre de variable.

3Propiedades (Atributos)

Son las características del objeto y se representan a través de variables. Puedes preguntarte: ¿qué características definen a una persona? Las características serían el nombre, el apellido, la edad, etc.

Para definir atributos, simplemente creas una variable: primero el tipo de dato, luego el nombre de la variable. Puedes inicializarla o no, y al final no olvides el punto y coma.

Ejemplo:

String nombre;
int edad = 30;

De esta forma hemos creado las propiedades de una clase.

4Métodos

Para crear un método es igual que crear una función. Lo creamos de la siguiente forma:

public void mostrarEdad() {
    System.out.println("Edad: " + edad);
}

// En la función principal
persona.mostrarEdad();

// Resultado:
// Edad: 45

5¿Qué es un Objeto?

Un objeto es una instancia de una clase que tiene acceso a las propiedades y métodos definidos en ella.

🎯 Analogía

Piensa en la clase como un molde de galletas y el objeto como cada galleta individual que produces con ese molde. Todas las galletas tienen la misma forma (clase), pero cada una puede tener diferentes decoraciones (valores de atributos).

6Cómo Crear un Objeto

Persona persona = new Persona();

Desglosemos esta línea:

  • Persona → Nombre de la clase (tipo de dato)
  • persona → Nombre del objeto (variable)
  • = → Operador de asignación
  • new → Palabra reservada para crear el objeto en memoria
  • Persona() → Constructor de la clase

💡 Nota: Aunque no hayamos definido un constructor, Java invoca automáticamente el constructor vacío por defecto.

7Acceder a Propiedades y Métodos

Para acceder a las propiedades y métodos del objeto utilizamos el operador punto .

System.out.println(persona.edad);
// Imprime: 30

Las propiedades funcionan como variables, por lo que podemos reasignar valores:

Persona persona = new Persona();
persona.edad = 45;
System.out.println(persona.edad);
// Imprime: 45

8Entrada de Datos por Teclado

Primero debes importar la clase Scanner:

import java.util.Scanner;

Luego creas un objeto de la clase Scanner:

Scanner sc = new Scanner(System.in);
System.out.println("Ingrese su apellido: ");
String apellido = sc.nextLine();
System.out.println("Apellido ingresado: " + apellido);

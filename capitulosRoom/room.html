Como crear una base de datos utilizando room

room es una libreria de Android que se utiliza para la persistencia de datos

room tiene tres partes principales

Entity

representa una tabla en la base datos donde cada propiedad de la data class es una columna de la tabla si no utilizas la anotacion @Entity room no sabra que clase debe convertir a una tabla 


    @Entity (tableName = "notes")
data class Note (
    @PrimaryKey(autoGenerate = true)
    val id:Int=0,
    val title:String="",
    val description:String=""
)

Dao (DATA ACCESS OBJECT)

es la capa que se encarga de comunicarse con la base de datos 


    @Dao
interface NotesDao {
    @Insert(onConflict = OnConflictStrategy.REPLACE)
    suspend fun insertNote(note: Note)

    @Delete
    suspend fun delete(note: Note)

    @Update
    suspend fun updateNote(note: Note)

    @Query("select * from notes")
    fun getAllNotes(): Flow>
}

Database


    @Database(entities = [Note::class], version = 1)
abstract class NoteDatabase:RoomDatabase() {
    abstract fun getDao():NotesDao
}

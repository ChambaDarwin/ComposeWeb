Navegación · NavGraph

Navigation con NavController

NavController gestiona la navegación entre pantallas, permite navegar entre ellas, pasar parámetros y manejar el backStack.

KOTLIN · NavGraph.kt 
@Composable
fun NavGraphNotes() {
    val navController = rememberNavController()
    NavHost(
        navController = navController,
        startDestination = Rutas.Main.ruta
    ) {
        composable(Rutas.Main.ruta) {
            NotasScreen {
                navController.navigate(Rutas.Add.ruta)
            }
        }
        composable(Rutas.Add.ruta) {
            AddNotesScreen()
        }
    }
}
Galería · ActivityResult

Seleccionar fotos de galería

Usamos rememberLauncherForActivityResult para abrir cualquier actividad del sistema. Dependiendo del contrato puedes abrir la galería, tomar fotos, solicitar permisos, etc.

KOTLIN · Compose 
val launcher = rememberLauncherForActivityResult(
    contract = ActivityResultContracts.GetMultipleContents()
) { uris ->
    // uris: List<Uri> con las imágenes seleccionadas
}

// Botón que abre la galería
Type.GALERY -> {
    launcher.launch("image/*")
}

Realizaremos una app de agenda de notas con dos pantallas: una para mostrar todas las notas guardadas y otra para registrar nuevas notas usando Room como base de datos local.

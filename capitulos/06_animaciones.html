🎨 Animaciones Lottie

Dale vida a tu app con animaciones profesionales en Jetpack Compose

✨ ¿Qué son las animaciones Lottie?

Lottie es una biblioteca desarrollada por Airbnb que permite renderizar animaciones de After Effects en tiempo real. Son archivos ligeros en formato JSON que ofrecen animaciones de calidad profesional sin comprometer el rendimiento de tu aplicación.

📦 Paso 1: Instalación de Dependencias

1
Agregar la dependencia de Lottie

Abre el archivo build.gradle.kts (Module: app) y agrega la siguiente dependencia en el bloque dependencies:

build.gradle.kts (Module: app)
dependencies {
    implementation("com.airbnb.android:lottie-compose:6.4.0")
    
    // Resto de tus dependencias...
}
💡 Consejo: Después de agregar la dependencia, haz clic en "Sync Now" en la barra superior de Android Studio para sincronizar el proyecto.

📁 Paso 2: Preparar tus Animaciones

2
Descargar y organizar archivos Lottie

¿Dónde conseguir animaciones Lottie?

  • LottieFiles: lottiefiles.com - Miles de animaciones gratuitas
  • Iconscout: Animaciones premium y gratuitas
  • Crear propias: Exporta desde After Effects con el plugin Bodymovin

Ubicación del archivo:

  1. Descarga tu animación en formato .json
  2. En Android Studio, navega a: app/src/main/res/
  3. Crea la carpeta raw si no existe (clic derecho → New → Android Resource Directory → Resource type: raw)
  4. Copia tu archivo .json dentro de res/raw/
⚠️ Importante: El nombre del archivo debe estar en minúsculas y sin espacios. Ejemplo: loading_animation.json en lugar de Loading Animation.json

💻 Paso 3: Implementación en Compose

3
Crear la función composable

Crea una función composable reutilizable que muestre la animación Lottie:

LottieAnimationScreen.kt
import androidx.compose.runtime.Composable
import androidx.compose.runtime.getValue
import androidx.compose.ui.Modifier
import com.airbnb.lottie.compose.*

@Composable
fun LottieAnimationScreen() {
    // Cargar la composición de la animación
    val composition by rememberLottieComposition(
        LottieCompositionSpec.RawRes(R.raw.mi_animacion)
    )
    
    // Controlar el progreso de la animación
    val progress by animateLottieCompositionAsState(
        composition = composition,
        iterations = LottieConstants.IterateForever
    )
    
    // Renderizar la animación
    LottieAnimation(
        composition = composition,
        progress = { progress },
        modifier = Modifier.size(200.dp)
    )
}
4
Entendiendo el código

🔍 Desglose de componentes:

1. rememberLottieComposition:

  • Carga el archivo JSON de la animación
  • remember evita que se recargue en cada recomposición
  • LottieCompositionSpec.RawRes(R.raw.mi_animacion) especifica la ubicación del archivo

2. animateLottieCompositionAsState:

  • Controla el progreso y reproducción de la animación
  • composition: La animación cargada
  • iterations: Número de repeticiones

3. LottieAnimation:

  • Componente que renderiza la animación en pantalla
  • composition: La animación a mostrar
  • progress: Estado actual de la animación (0.0 a 1.0)
  • modifier: Personalización de tamaño y estilo

⚙️ Paso 4: Opciones de Configuración

5
Personalizar el comportamiento

Opciones de repetición:

// Reproducir una sola vez
iterations = 1

// Reproducir infinitamente
iterations = LottieConstants.IterateForever

// Reproducir 5 veces
iterations = 5

Control de velocidad:

val progress by animateLottieCompositionAsState(
    composition = composition,
    iterations = LottieConstants.IterateForever,
    speed = 2f  // 2x más rápido (0.5f = mitad de velocidad)
)

Reproducción automática:

val progress by animateLottieCompositionAsState(
    composition = composition,
    isPlaying = true,  // false para pausar
    restartOnPlay = true  // Reiniciar al reproducir
)

🎯 Ejemplo Completo Avanzado

Ejemplo con control manual de reproducción:

AdvancedLottieAnimation.kt
@Composable
fun AdvancedLottieAnimation() {
    var isPlaying by remember { mutableStateOf(true) }
    
    val composition by rememberLottieComposition(
        LottieCompositionSpec.RawRes(R.raw.loading_animation)
    )
    
    val progress by animateLottieCompositionAsState(
        composition = composition,
        isPlaying = isPlaying,
        iterations = LottieConstants.IterateForever,
        speed = 1.5f
    )
    
    Column(
        modifier = Modifier.fillMaxSize(),
        horizontalAlignment = Alignment.CenterHorizontally,
        verticalArrangement = Arrangement.Center
    ) {
        LottieAnimation(
            composition = composition,
            progress = { progress },
            modifier = Modifier
                .size(300.dp)
                .padding(16.dp)
        )
        
        Button(onClick = { isPlaying = !isPlaying }) {
            Text(if (isPlaying) "Pausar" else "Reproducir")
        }
    }
}

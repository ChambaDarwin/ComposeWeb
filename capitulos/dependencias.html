📦 Dependencias en Android

Guía completa con Jetpack Compose, Retrofit, Hilt, Firebase y más

🚀 ¿Qué encontrarás aquí?

Una referencia completa de todas las dependencias necesarias para desarrollar aplicaciones Android modernas. Incluye configuración de Retrofit, Dagger Hilt, Firebase Realtime Database, testing con MockK y más.

📱 Dependencias del módulo app (build.gradle.kts)
🔄 Lifecycle & ViewModel
Para manejar el ciclo de vida y ViewModels en Compose
// Lifecycle ViewModel Compose
implementation("androidx.lifecycle:lifecycle-viewmodel-compose:2.8.0")
🌐 Retrofit & Networking
Cliente HTTP para consumir APIs REST
// Retrofit + Conversor Gson
implementation("com.squareup.retrofit2:retrofit:2.9.0")
implementation("com.squareup.retrofit2:converter-gson:2.9.0")
📄 Paginación
Para cargar datos de forma paginada en listas
// Paging 3 con Compose
implementation("androidx.paging:paging-compose:3.3.0")
💉 Dagger Hilt
Inyección de dependencias para Android
// Hilt completo
// Hilt principal
implementation("com.google.dagger:hilt-android:2.51.1")
kapt("com.google.dagger:hilt-compiler:2.51.1")

// Para inyectar ViewModels en Compose
implementation("androidx.hilt:hilt-navigation-compose:1.2.0")

// (Opcional) Para pruebas con Hilt
androidTestImplementation("com.google.dagger:hilt-android-testing:2.51.1")
kaptAndroidTest("com.google.dagger:hilt-compiler:2.51.1")
🔥 Firebase Realtime Database
Base de datos en tiempo real de Firebase
// Firebase Database
implementation(platform("com.google.firebase:firebase-bom:33.7.0"))
implementation("com.google.firebase:firebase-database-ktx")
🧪 Testing con MockK
Framework de mocking para pruebas unitarias en Kotlin
// MockK Testing
testImplementation("io.mockk:mockk:1.13.10")

// Para testing de coroutines
testImplementation("org.jetbrains.kotlinx:kotlinx-coroutines-test:1.7.3")
🔌 Plugins del módulo app (build.gradle.kts)
plugins { ... }
plugins {
    id("com.android.application")
    id("org.jetbrains.kotlin.android")
    id("com.google.dagger.hilt.android")
    id("kotlin-kapt")
    id("com.google.gms.google-services") // Para Firebase
}
🏗️ Plugins a nivel del proyecto (build.gradle.kts raíz)
plugins { ... }
plugins {
    id("com.android.application") version "8.2.0" apply false
    id("org.jetbrains.kotlin.android") version "1.9.20" apply false
    id("com.google.dagger.hilt.android") version "2.51.1" apply false
    id("com.google.gms.google-services") version "4.4.0" apply false
}
⚙️ Configuración adicional Firebase
📝 Archivo google-services.json
Descarga el archivo desde la consola de Firebase y colócalo en la carpeta app/
Ubicación del archivo
tu-proyecto/
├── app/
│   ├── google-services.json  ← Aquí va el archivo
│   └── build.gradle.kts
└── build.gradle.kts

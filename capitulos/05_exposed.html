📱 ExposedDropdownMenuBox

El equivalente moderno al Spinner en Jetpack Compose

💡 ¿Qué aprenderás en este tutorial? Descubrirás cómo implementar el componente ExposedDropdownMenuBox, el reemplazo del tradicional Spinner de XML en Jetpack Compose. Aprenderás a crear menús desplegables elegantes y funcionales paso a paso.

Si vienes del mundo de XML en Android, este componente es el equivalente moderno al Spinner. Te permite desplegar una lista de opciones para que el usuario seleccione una de ellas. El potencial de este componente es enorme y su implementación es más declarativa y flexible que su predecesor.

🎯 Configuración Inicial

1 Crear la función composable

Primero, crearemos una función composable reutilizable que contendrá:

  • expanded: Estado boolean que indica si el menú está desplegado
  • text: Estado que almacena el valor seleccionado
  • lista: Lista con las opciones disponibles
@OptIn(ExperimentalMaterial3Api::class)
@Composable
fun ExposedDropdownUi() {
    val lista = listOf("Cliente", "Administrador")
    var expanded by rememberSaveable { mutableStateOf(false) }
    var text by rememberSaveable { mutableStateOf("") }
}
2 Configurar el contenedor ExposedDropdownMenuBox

Este contenedor envuelve tanto el TextField como el menú desplegable. Recibe dos parámetros obligatorios:

  • expanded: Valor boolean que controla la visibilidad del menú
  • onExpandedChange: Lambda que se ejecuta cuando cambia el estado

Cuando el usuario presiona el contenedor, el parámetro it es true, lo que actualiza el estado y despliega las opciones.

ExposedDropdownMenuBox(
    expanded = expanded,
    onExpandedChange = { expanded = it }
) {
    // Aquí van el TextField y el menú
}

🎨 Configuración del TextField

3 Implementar el TextField

El TextField muestra el valor seleccionado. Configuración importante:

  • value: Muestra el texto seleccionado actualmente
  • onValueChange: Se deja vacío porque no se usa entrada de teclado
  • readOnly: Se establece en true para evitar edición manual
  • trailingIcon: Muestra el ícono de flecha indicadora
  • menuAnchor(): Modificador esencial que vincula el TextField con el menú
OutlinedTextField(
    value = text,
    onValueChange = {},
    readOnly = true,
    label = { Text("Seleccionar rol") },
    trailingIcon = {
        ExposedDropdownMenuDefaults.TrailingIcon(expanded = expanded)
    },
    modifier = Modifier
        .fillMaxWidth()
        .menuAnchor()
        .padding(16.dp)
)

📋 Configuración del Menú Desplegable

4 Implementar ExposedDropdownMenu

Este componente muestra las opciones disponibles. Parámetros clave:

  • expanded: Controla la visibilidad del menú
  • onDismissRequest: Se ejecuta cuando el usuario toca fuera del menú
ExposedDropdownMenu(
    expanded = expanded,
    onDismissRequest = { expanded = false }
) {
    lista.forEach { opcion ->
        DropdownMenuItem(
            text = { Text(opcion) },
            onClick = {
                text = opcion
                expanded = false
            }
        )
    }
}

Cada DropdownMenuItem representa una opción. Al hacer clic:

  1. Actualiza el valor del TextField con la opción seleccionada
  2. Cierra el menú estableciendo expanded en false

✅ Código Completo

@OptIn(ExperimentalMaterial3Api::class)
@Composable
fun ExposedDropdownUi() {
    val lista = listOf("Cliente", "Administrador")
    var expanded by rememberSaveable { mutableStateOf(false) }
    var text by rememberSaveable { mutableStateOf("") }

    ExposedDropdownMenuBox(
        expanded = expanded,
        onExpandedChange = { expanded = it }
    ) {
        OutlinedTextField(
            value = text,
            onValueChange = {},
            readOnly = true,
            label = { Text("Seleccionar rol") },
            trailingIcon = {
                ExposedDropdownMenuDefaults.TrailingIcon(expanded = expanded)
            },
            modifier = Modifier
                .fillMaxWidth()
                .menuAnchor()
                .padding(16.dp)
        )

        ExposedDropdownMenu(
            expanded = expanded,
            onDismissRequest = { expanded = false }
        ) {
            lista.forEach { opcion ->
                DropdownMenuItem(
                    text = { Text(opcion) },
                    onClick = {
                        text = opcion
                        expanded = false
                    }
                )
            }
        }
    }
}

Cómo crear un selector de hora

Guía paso a paso para implementar TimePickerDialog en Jetpack Compose y convertir el resultado a texto legible.

1
Crear la función composable

El primer paso es crear una función composable que reciba como parámetro una función lambda. Esta lambda acepta un valor de tipo Long porque la hora seleccionada se devuelve en milisegundos, lo que facilita su manipulación y conversión posterior. 

@Composable
fun SelectHour(context: Context, time: (Long) -> Unit) {
}
2
Configurar el TimePickerDialog

A continuación, se crea una instancia de TimePickerDialog. Esta clase recibe cinco parámetros:

1) El contexto de la aplicación.
2) Una lambda que se ejecuta cuando el usuario confirma la selección. Sus parámetros son la vista, la hora y los minutos — aunque la vista raramente se usa, se representa con _.
3) La hora inicial que mostrará el selector.
4) Los minutos iniciales.
5) Un booleano: true para formato de 24 horas, false para formato de 12 horas (AM/PM). 

@Composable
fun SelectHour(context: Context, time: (Long) -> Unit) {
    val calendar = Calendar.getInstance()
    val dialog = TimePickerDialog(
        context,
        { _, hour, minute ->

        },
        calendar.get(Calendar.HOUR_OF_DAY),
        calendar.get(Calendar.MINUTE),
        false
    )
}
3
Asignar la hora seleccionada al calendario

Dentro de la lambda, se actualiza el objeto Calendar con los valores que el usuario eligió. Para ello se usa el método set(), que recibe dos argumentos: el campo del calendario a modificar (por ejemplo, HOUR_OF_DAY o MINUTE) y el nuevo valor. Finalmente, se invoca la lambda con el tiempo resultante en milisegundos. 

{ _, hour, minute ->
    calendar.set(Calendar.HOUR_OF_DAY, hour)
    calendar.set(Calendar.MINUTE, minute)
    time(calendar.timeInMillis)
},
4
Mostrar el diálogo

Por último, se agrega un botón. Al presionarlo, se llama al método show() del diálogo para mostrarlo en pantalla. 

Button(onClick = { dialog.show() }) {
    Text("Seleccionar hora")
}

Cómo convertir milisegundos a texto legible

Una vez obtenida la hora en milisegundos, es necesario mostrarla en un formato comprensible para el usuario. Para esto se utiliza la clase SimpleDateFormat, que permite transformar valores de tiempo en cadenas de texto con un patrón personalizado.

Nota: Usar Locale.getDefault() como segundo parámetro garantiza que el formato respete el idioma y la región configurados en el dispositivo del usuario. 
val formato = SimpleDateFormat("HH:mm", Locale.getDefault())
// Obtiene el idioma y la región del dispositivo

Luego se emplea la clase Date para representar un instante específico en el tiempo. Al pasarle un valor en milisegundos como argumento, el objeto encapsula la fecha y hora correspondiente.

Finalmente, el método format() recibe ese objeto Date y lo convierte a texto aplicando el patrón definido. 

val formato = SimpleDateFormat("HH:mm", Locale.getDefault())
horaTexto = formato.format(Date(milisegundos))
Patrón HH:mm: HH representa las horas en formato de 24 horas (00–23), mientras que mm representa los minutos (00–59). Si prefieres formato de 12 horas, usa hh:mm a.

📸 Seleccionar y Almacenar Imágenes

Guía completa para Android con Jetpack Compose

En este tutorial aprenderás a seleccionar imágenes desde la galería, almacenarlas en el directorio interno de la aplicación y mostrarlas correctamente en tu app Android.

🔧 Dependencias Necesarias

Añade la siguiente dependencia en el archivo build.gradle del módulo de tu aplicación:

implementation("io.coil-kt:coil-compose:2.6.0")

📂 Activity Result Launcher

Para abrir la galería del teléfono y seleccionar archivos, utilizamos rememberLauncherForActivityResult:

val launcher = rememberLauncherForActivityResult(
    contract = ActivityResultContracts.GetContent()
) { uri: Uri? ->
    // Aquí manejas la URI de la imagen seleccionada
}

Este componente nos permite abrir actividades del sistema para realizar acciones como:

  1. Abrir la cámara
  2. Seleccionar archivos de la galería
  3. Solicitar permisos al usuario
  4. Capturar fotografías

Recibe como parámetro un contrato que define qué se enviará a la actividad y qué resultado se obtendrá.

📋 Contratos Disponibles

Contrato Descripción
TakePicture() Captura una foto utilizando la cámara del dispositivo
GetContent() Permite elegir cualquier tipo de archivo (imagen, PDF, etc.)
PickVisualMedia() Selector optimizado para imágenes y videos
RequestPermission() Solicita permisos específicos al usuario

🖼️ Mostrar una Imagen con AsyncImage

Para mostrar la imagen seleccionada, utiliza el componente AsyncImage de Coil:

AsyncImage(
    model = uri,
    contentDescription = "Imagen seleccionada",
    modifier = Modifier
        .fillMaxWidth()
        .height(300.dp)
)

💾 Convertir URI a Archivo Físico

Para almacenar permanentemente la imagen en el almacenamiento interno de la aplicación, sigue estos pasos:

1. Leer la imagen desde su ubicación original:

val inputStream = context.contentResolver.openInputStream(uri)!!

2. Crear un archivo en el directorio privado de la app:

El objeto File representa la ubicación donde se guardará la imagen. context.filesDir es una carpeta privada que Android asigna exclusivamente a tu aplicación.

val file = File(context.filesDir, "${System.currentTimeMillis()}.jpg")

3. Crear un flujo de salida para escribir los datos:

val outputStream = FileOutputStream(file)

4. Copiar los bytes y cerrar los flujos:

inputStream.copyTo(outputStream)
inputStream.close()
outputStream.close()
return file.absolutePath
💡 Nota: Este proceso copia físicamente la imagen al almacenamiento interno de tu aplicación, permitiendo acceder a ella incluso si el archivo original es eliminado.

👁️ Mostrar la Imagen Almacenada

1. Crear un objeto File con la ruta:

Este objeto apunta a la ubicación física del archivo en el sistema de archivos de Android.

val file = File(ruta)

2. Validar existencia y decodificar la imagen:

if (file.exists()) {
    val bitmap = BitmapFactory.decodeFile(file.absolutePath)
    bitmap?.let {
        Image(
            bitmap = it.asImageBitmap(),
            contentDescription = "Imagen almacenada",
            modifier = Modifier.fillMaxWidth()
        )
    }
}

El método asImageBitmap() convierte el bitmap al formato que utiliza Jetpack Compose.

📋 Listar Todas las Imágenes Almacenadas

Para obtener todos los archivos guardados en el directorio privado:

context.filesDir.listFiles()?.forEach { file ->
    println("Archivo: ${file.name}")
}

🗑️ Eliminar Imágenes

Para eliminar todas las imágenes almacenadas:

context.filesDir.listFiles()?.forEach { file ->
    file.delete()
}

Para eliminar un archivo específico:

val file = File(ruta)
if (file.exists()) {
    file.delete()
}

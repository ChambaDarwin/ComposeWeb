Propiedades de Text y TextField

Referencia completa para Jetpack Compose

📝 Componente Text

Propiedad Tipo Descripción Ejemplo
text String Texto a mostrar text = "Hola"
fontSize TextUnit Tamaño de la fuente fontSize = 20.sp
fontWeight FontWeight Grosor del texto fontWeight = Bold
fontStyle FontStyle Estilo (normal o itálica) fontStyle = Italic
color Color Color del texto color = Color.Red
textAlign TextAlign Alineación del texto textAlign = Center
lineHeight TextUnit Altura entre líneas lineHeight = 24.sp
maxLines Int Número máximo de líneas maxLines = 2
overflow TextOverflow Comportamiento del desbordamiento overflow = Ellipsis
letterSpacing TextUnit Espaciado entre letras letterSpacing = 0.5.sp
textDecoration TextDecoration Decoración (underline, lineThrough) textDecoration = Underline

✏️ Componente TextField

Propiedad Tipo Descripción Ejemplo
value String Valor actual del campo value = text
onValueChange Lambda Callback al cambiar el texto onValueChange = { text = it }
label @Composable Etiqueta descriptiva label = { Text("Nombre") }
placeholder @Composable Texto de ayuda cuando está vacío placeholder = { Text("Escribe...") }
leadingIcon @Composable Icono al inicio del campo leadingIcon = { Icon(...) }
trailingIcon @Composable Icono al final del campo trailingIcon = { Icon(...) }
singleLine Boolean Limita a una sola línea singleLine = true
maxLines Int Número máximo de líneas maxLines = 3
enabled Boolean Habilita o deshabilita el campo enabled = true
readOnly Boolean Campo de solo lectura readOnly = false
keyboardOptions KeyboardOptions Opciones del teclado (tipo, acción) keyboardOptions = KeyboardOptions(keyboardType = Number)
keyboardActions KeyboardActions Acciones del teclado keyboardActions = KeyboardActions(onDone = {...})
visualTransformation VisualTransformation Transformación visual (ej: password) visualTransformation = PasswordVisualTransformation()
isError Boolean Indica estado de error isError = false
colors TextFieldColors Colores personalizados colors = TextFieldDefaults.colors(...)

🔧 Modifiers Comunes

Modifier Descripción Ejemplo
padding Añade espacio interno .padding(16.dp)
fillMaxWidth Ocupa todo el ancho disponible .fillMaxWidth()
width Define ancho específico .width(200.dp)
height Define altura específica .height(56.dp)
background Color de fondo .background(Color.White)
border Añade borde .border(2.dp, Color.Gray)
clip Recorta con forma específica .clip(RoundedCornerShape(8.dp))
clickable Hace el elemento clickeable .clickable { /* acción */ }
alpha Transparencia .alpha(0.5f)
shadow Añade sombra .shadow(4.dp)

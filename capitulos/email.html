Como crear un TextField para visualizar la contraseña

en capitulos anterios ya aprendimos a trabajar con el componente TextField puedes revisar aqui vamos a agregar mas propiedades leadingIcon y trailingIcon son funciones lambdas y las vamos a implementar con leadingIcon podemos crear un icon en la derecha y con leadingIcon en la parte izquierda dentro de la funciones podemos crear un icono


   @Composable
fun TextFieldPassword(
    value: String,
    onValueChange: (String) -> Unit

) {
    var show by rememberSaveable { mutableStateOf(false) }

    OutlinedTextField(
        visualTransformation =if(show) VisualTransformation .None else
            PasswordVisualTransformation(),
        leadingIcon = {
            Icon(
                painter = painterResource(R.drawable.ic_lock),
                contentDescription = ""
            )
        },
        trailingIcon = {
            IconButton(onClick = {show = !show}) {
                Icon(
                    painter = painterResource(if(show) R.drawable.eyes_close else R.drawable.eyes_open),
                    contentDescription = ""
                )
            }
        }
    )
}

Explicacion

  1. PasswordVisualTransformation() convierte visualmente el texto a caracteres
  2. VisualTransformation.None muestra el texto real
  3. cuando precione el icono del ojo se mostrara el texto real
